Notes  
   

MOFcart has some robust optional parameters for handling PopUp windows. There are conditions for closing the window automatically at timed interval, for formatting product added to cart confirmation, for sending custom messages, for bypassing the Main ViewCart screen & redirection options.

If you do anything with PopUps you should read through the documentation section on this. PopUps that work simultaneously with the Main MOFcart screen, or even a Main site screen, must be Named and targetted correctly. The Pop Up examples used in this package also rely on javaScript for some of the features. It is important that you get all the embedded javaScript and/or events in place between the various windows for everything to work correctly. You will need to make yourself familiar with all the concepts in the documentation for PopUp Windows.
