Usage  
   

You place buy buttons next to individual items around your web site and require quantity and/or other product options.

Instructions: Select Size, Color, and Band Style and click the Buy button. The Western Hat will be added to your basket
 

Example  
   

The Classic Western Hat $ 45.00


Size
Color
Band


 

Syntax  
   

<FORM method=POST action="http://www.gardenthemes.com/cgi-bin/mcart/mof.cgi">
<input type="hidden" name="postmode" value="SINGLEPOST">
<input type="hidden" name="quantity" value="1">

<input type="hidden" name="order" value="Western Hat----[img align=left vspace=18 hspace=6 src=http://www.gardenthemes.com/mofcart/images/westernhat_sm.gif width=50 height=40 border=0] [li] The Classic Western Hat ----45.00----2----1">


<SELECT name="HatSize" size="1">
<OPTION selected value="">Select
<OPTION value="Small">Small
<OPTION value="Medium">Medium
<OPTION value="Large">Large
<OPTION value="Extra Large">Extra Large
</SELECT>

<SELECT name="HatColor" size="1">
<OPTION selected value="">Select
<OPTION value="Brown">Brown
<OPTION value="Gray">Gray
<OPTION value="White">White
<OPTION value="Black">Black
</SELECT>

<SELECT name="HatBand" size="1">
<OPTION selected value="">Select
<OPTION value="Snake Skin">Snake Skin
<OPTION value="Rabbbit Tail">Rabbit Tail
<OPTION value="Fly Fishing">Fly Fishing
<OPTION value="Bear Claw">Bear Claw
</SELECT>

<input type="image" src="newbuybtn.gif" width=51 height=18 border=0
alt="Click Here to Add To Cart">
</FORM>
 

Notes  
   

This submits one product with three required product options: Size, Color, and Band

Note the main product is submitted as hidden input. Also, note the use of HTML [ pseudo tags ] for sending the image to the cart

All three product option field names must be defined in <mof.conf>

  • First in %product_fields
  • And (if required) in @field_validation

Note the default selected "" (null) in dropboxes for validation accuracy
