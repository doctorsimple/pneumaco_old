Usage  
   

I don't know. Someone asked if it could be done. Here it is.

Instructions: Select items in the grid to be placed into the cart by entering the Quantity.
Click the Add To Cart button, and your selections will be added to the cart.

Syntax  
   

<FORM method=POST action="http://www.gardenthemes.com/cgi-bin/mcart/mof.cgi">
<INPUT type="hidden" name="postmode" value="QUANTITYBOXES">

<INPUT type="hidden" name="order1" value="ITEM 1----DESCRIPTION----12.99----1----1">
<INPUT type="hidden" name="order2" value="ITEM 2----DESCRIPTION----12.99----1----1">
<INPUT type="hidden" name="order3" value="ITEM 3----DESCRIPTION----12.99----1----1">
<INPUT type="hidden" name="order4" value="ITEM 4----DESCRIPTION----12.99----1----1">

<input maxlength="5" Class="SmallBoxFormat" size="2" name="quantity1">
<input maxlength="5" Class="SmallBoxFormat" size="2" name="quantity2">
<input maxlength="5" Class="SmallBoxFormat" size="2" name="quantity3">
<input maxlength="5" Class="SmallBoxFormat" size="2" name="quantity4">

<input name="update_btn" type=submit value="Add To Cart">
<input name="reset_btn" type=reset value="Reset">
</FORM>
 

Notes  
   
  • All 54 items are submitted as hidden input as "orderN"
  • Only items with corresponding "quantityN" input will be posted to cart
  • This example has no product options
