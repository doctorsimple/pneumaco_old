Another Pop Up Example:

This is just a variation on the PopUp 3 example, which shows how you can manipulate the outcome of "add to cart". It times the Added to cart message for 1 sec. then redirects to the ViewCart Page. You can also redirect to any Catalogue page, etc.

Make note of how the Params are adjusted for POP_VIEWCART, POP_AUTOCLOSE, REDIRECT_URL, POP_TARGET. Especially note that REDIRECT_URL requires additional embedded Params in the string passed to <mof.cgi> for ViewCart triggers. Consult the Documentation > Using Popup Windows
