Usage  
   

For the impulsive buyer who doesn't have time to click nine items. Just swipe all nine items and click Add to cart.

Instructions: Use any Windows multiple select method - swipe all items while holding your left mouse button, or hold the Control and Left Click any item in the list.Click the add to cart button. Those items will be added to your basket.

 

Example  
   



 

Syntax  
   

<FORM method=POST action="http://www.gardenthemes.com/cgi-bin/mcart/mof.cgi">
<INPUT type="hidden" name="postmode" value="CHECKBOXES">

<SELECT MULTIPLE SIZE=9 NAME=order>
<OPTION VALUE="1----Box Baseball Cards----24 Baseball Cards----10.00----0.11----1">
Box Baseball Cards
<OPTION VALUE="1----Second Nature Books----10 Children books----65.00----0.22----0">
Second Nature Books
<OPTION VALUE="1----Each Day Poster # 1----Mountain scene poster----11.50----0.33----1">
Each Day Poster # 1
<OPTION VALUE="1----Each Day Poster # 34----Lake scene poster----11.50----0.44----1">
Each Day Poster # 34
<OPTION VALUE="1----Today In History v1.0----History software----21.65----0.55----1">
Today In History v1.0
<OPTION VALUE="1----New York Times----Subscription to the NYT----19.25----0.66----1">
New York Times
<OPTION VALUE="1----Magic Box Gifts----Surprise gift box----12.25----0.77----1">
Magic Box Gifts
<OPTION VALUE="2----The Finish Line----Autobiography by Bill Waters----31.99----0.88----1">
The Finish Line
<OPTION VALUE="3----The Naked Kingdom----Confessions by Monica L.----50.95----0.99----1">
The Naked Kingdom
</SELECT>

<input type=submit value="Add To Cart">
<input type=reset value="Reset Form">
</FORM>
 

Notes  
   

This sets up a multi select list box using CHECKBOXES mode.

Note the order data is simply submitted as the OPTION value. When the add to cart button is clicked, all selected items are added to the cart

No product options are allowed for this type input

Note that quantity is submitted as the first field placeholder in the order input
value="Qty----Item----Desc----Price----ShipCode----TaxCode"
