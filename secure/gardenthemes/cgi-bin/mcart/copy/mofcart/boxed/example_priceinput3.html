Notes

This is a medley of Price Input that can be used with QUANTITYBOXES mode.

Note the hidden input for Source ONE/TWO "order" has Price, ShipCode, and TaxCode set to 0
.. denoting a zero base price, no shipping, and tax free status
.. you could set these to other settings as needed

Note the hidden input for Source THREE "order" has a preset amount of $ 25.00
.. and no shipping, tax free status

Note that both types of price input settings are enabled in <mof.conf>

  • The main price setting: $price_input
  • Declared User Input: %product_fields
  • Declared User Input:@price_input_adjustments
  • Declared User Input:@field_validation (if required)

Note you can allow zero based input by enabling $allow_zeros in <mof.conf>
