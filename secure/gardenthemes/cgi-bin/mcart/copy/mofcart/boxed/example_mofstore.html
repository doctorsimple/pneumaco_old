Main Store Portal  

This is an example placeholder page for the package installation. The default installation has a Catalogue button enabled which is sort of a "home" button, and provides a quick link back to a central order starting place. The default install setting for this is set to go to your site's home index page. You can set the Catalogue button to point to any page -- see settings in both the Front & Back End configuration files <mof.conf> & <mofpay.conf> for:
$menu_home_top and $menu_home_bottom

Also, when friendly error messages are enabled, MOF can provide a link to a main catalogue page, or main store portal, index, etc. Sort of a : quickest way back to the store in case of an error message. You can direct MOF to set this link to any page on your site. The setting is in the <mof.conf> configuration file called : $ErrMsgViewCart
