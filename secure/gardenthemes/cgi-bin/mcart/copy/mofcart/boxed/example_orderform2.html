Use: You offer items that fit nicely on a simple, one-page order form and do not require quantity, size, color, or other options. This example uses nifty little pop-up windows to display more information about the product.

Instructions: Check the items you wish to purchase, and click the add to cart button. Those items will be added to your basket.

FORM syntax ..  

<FORM method=POST action="http://www.gardenthemes.com/cgi-bin/mcart/mof.cgi">
<INPUT type="hidden" name="postmode" value="CHECKBOXES">

<input type="checkbox" name="order"
value="1----Box Baseball Cards----24 Collectable Baseball Cards----10.00----1----1">

<input type="checkbox" name="order"
value="1----Second Nature Books----Collection of 10 Children books----65.00----1----1">

<input type="checkbox" name="order"
value="1----Each Day Poster # 1----Mountain scene poster----11.50----1----1">

<input type=submit value="Add To Cart">
</FORM>

Notes

This sets up rows of checkbox input using CHECKBOXES mode.

Note the order data is simply submitted as the value of the checkbox. When the add to cart button is clicked, all checked items are added to the cart

No product options are allowed for this type input

Note that quantity is submitted as the first field placeholder in the order input
value="Qty----Item----Desc----Price----ShipCode----TaxCode"
