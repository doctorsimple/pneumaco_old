FORM syntax ..  

<FORM action=http://www.gardenthemes.com/cgi-bin/mcart/mof.cgi method=post>
<input type=hidden name=postmode value=SINGLEPOST>

<SELECT name=order size="1">
<OPTION selected value="">Select License Type

<OPTION
value="A96-01199----MS OFFICE PREMIUM 2004 32 BIT WIN SR1 (CD)----699.00----1----1">
Full New License CD-ROM $ 699.00

<OPTION
value="A96-01200----MS OFFICE PREMIUM 2004 32 BIT WIN COMP/VER UPG SR1 (CD)----424.00----1----1">
Competitive/Version Upgrade CD-ROM $ 424.00

</OPTION>
</SELECT>

<INPUT maxLength=3 name=quantity size=3>
<INPUT border=0 height=34 src="addtocart.gif" type=image width=45 name="Add To Cart">
</FORM>

Notes

This submits one product at a time, however, you can select the item from a dropbox, and enter the quantity before clicking the Add to cart button

The main product is submitted as <OPTION> value input.

Note the use of "" (null) as the default selection in the dropboxe. This is required for validation

If a quantity were not entered, then MOF would default to 1 for the item selected
