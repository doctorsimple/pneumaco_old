Notes

This submits a product with a base price, and five product options.

All five product option field names must be defined in <mof.conf>

  • First in %product_fields
  • And (if required) in @field_validation

The DrumSkin field name must also be defined in @field_adjustments because it is price adjusting.

Note the special format for the DrumSkin field value="Description-----0.00"

Note the default selected "" (null) in dropboxes for validation accuracy

Note that the radio button groups need a default checked for validation
