Price Input Field 1 Return to Examples

Use: You need the ability for the customer to enter a main price for a product or service..

Instructions: Enter the price for this service, and click the Add to Cart button. The item will be added to your cart.

Enter Amount of Donation
 

FORM syntax ..  

<FORM method=POST action="http://www.gardenthemes.com/cgi-bin/mcart/mof.cgi">
<INPUT type="hidden" name="postmode" value="SINGLEPOST">
<INPUT type="hidden" name="quantity" value="1">

<INPUT type="hidden" name="order"
value="Agency Donation----Tax FREE Donation for our Agency or Cause----0----0----0">

<INPUT type="text" name="PriceInput" size=15 value="25.00">

<input name="update_btn" type=submit value="Add To Cart">
</FORM>

Notes

This is how to submit price input that shows up as the item's main price.

Note the main product data is submitted as hidden input

Note that the name of the price input field is PriceInput
.. as defined in the <mof.conf> setting $price_input

Also note that the input has Price, ShipCode, and TaxCode set to 0
.. denoting that it will be have no shipping costs and be tax free

Note you can allow zero based input by enabling $allow_zeros in <mof.conf>
