Multiple Products with Options Return to Examples

Use: You can set up multiple products and/or associated options on a one-page order form using one add to cart button.

Instructions:

  1. Select the item(s) you want to purchase by entering a number (1, 2, etc.) in the Qty box.
  2. Select options for each selected product (size, color, band).
  3. Click the add to cart button to add all items to your cart.

Notes

This is one Form capable of submitting multiple products, with product options

Note that quantityN is text box input. When a quantity is entered for a product, then the orderN and optionsN are added to the cart for that particular quantityN

Note the main product is submitted as hidden input. Also, note the use of HTML [ pseudo tags ] for sending the image to the cart

All three product options Root_Field_Name must be defined in <mof.conf>.

  • First in %product_fields
  • And (if required) in @field_validation = ('HatSize','HatColor','HatBand')

Note the <mof.conf> settings use Root Field names only, the appended number is not included. The numbering scheme is only used in the Form input

Note the default selected "" (null) in dropboxes for validation accuracy
