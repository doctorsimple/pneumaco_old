Notes

This is how to append price input to an item's main price..

Note the hidden input for the main "order" has has Price, ShipCode, and TaxCode set to 0
.. denoting a zero base price, no shipping, and tax free status
.. you could set these to other settings as needed

Note the input field names must be declared in <mof.conf> as extra User Input:

  • First in %product_fields
  • Second in @price_input_adjustments
  • And (if required) in @field_validation

Note the extra fields are prefixed a, b, c for sorting

Note you can allow zero based input by enabling $allow_zeros in <mof.conf>
