Memberships Return to Examples

Use: This is an example of using extra input fields to collect Registration information for Memberships.

Instructions: Select Type of Membership and complete the required fields. Then click Add To Cart. You can add as many memberships to your cart as you want.

1. Select Membership Type
 
 
 
2. Complete Required Info
Name
Address
City, State, Zip
Name on Badge
Phone Number
E-Mail
 

FORM syntax ..  
<FORM method=POST action="http://www.gardenthemes.com/cgi-bin/mcart/mof.cgi">
<INPUT type="hidden" name="quantity" value="1">
<INPUT type="hidden" name="postmode" value="SINGLEPOST">

<select Class="BoxTwoFormat" size="1" name="order">
<option value="" selected>-- Please Select Type --</option>
<option value="[b]Adult Membership[/b]----[u]Registration Info[/u]:[br]----35.00----1----1">
Adult Membership</option>
<option value="[b]Companion Membership[/b]----[u]Registration Info[/u]:[br]----25.00----1----1">
Companion Membership</option>
<option value="[b]Teen Membership[/b]----[u]Registration Info[/u]:[br]----20.00----1----1">
Teen Membership</option>
<option value="[b]Child Membership[/b]----[u]Registration Info[/u]:[br]----15.00----1----1">
Child Membership</option>
</select>

<input size="50" name="memberAname">
<input size="50" name="memberBaddress">
<input size="50" name="memberCcsz">
<input size="50" name="memberDbadge">
<input size="50" name="memberEphone">
<input size="50" name="memberFemail">
<input size="50" name="memberGdob">

<input Class="ButtonFormat" type=submit value="Add To Cart">
<input Class="ButtonFormat" type=reset value="Reset Form">
</FORM>

Notes

This submits a Membership price, with required options for Member Info

Member Info field names must be defined in <mof.conf>

  • First in %product_fields
  • And (if required) in @field_validation

Note the default selected "" (null) in dropboxes for validation accuracy
