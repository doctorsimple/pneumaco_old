Quantity and Product Options 2 Return to Examples

Use: You offer products on a one-page order form and require quantity and/or other product options.

Instructions:

  1. Select the item(s) you want to purchase by entering a number (1, 2, etc.) in the Qty box.
  2. Select one option from each product option drop-down menu (size, color, band).
  3. Click the Buy button next to the item selected.
  4. Selected items will be added to your basket.

Notes

This is almost identical input as example_options1.html, except we have just lined up more items in OrderForm rows, and the quantity is text box input instead of hidden. Each product is a separate Form using SINGLEPOST mode.

Note the main product is submitted as hidden input. Also, note the use of HTML [ pseudo tags ] for sending the image to the cart

All three product option field names must be defined in <mof.conf>

  • First in %product_fields
  • And (if required) in @field_validation

Note the default selected "" (null) in dropboxes for validation accuracy
