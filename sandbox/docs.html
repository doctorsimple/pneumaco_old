This class alows you to "publish" and edit pages from wikipedia or other Mediawiki powered site without MediaWiki installation. The idea: Let say that you are a student and you study chemistry - you want to make your site about it. Using this class you can have your very own site with nice content + you can help wikipedia by adding/expanding its articles.

  • HOW TO USE
    • See the two examples



    CONSTRUCTOR
    $x = new wikipedia('PATCH TO WIKIPEDIA'); - where the PATCH is a link to a wikipedia or other mediawiki, like - http://en.wikipedia.org/

    METHODS:
    get_page($page, $header = false) - will show the content from $page (like: Content_Management_System), IF you set $header to non-false - HEAD section with encoding and Mediawiki CSS will be included)

    edit_page($page, $header = false) - the same as get_page, but this one will display the edit page with textarea
